# La Grande Révision

## Installation

- Créez un fork du repository
- Ouvrez le dossier avec Visual Studio Code
- Ouvrez un terminal dans Visual Studio Code
- Lancez la commande

```
npm install
```

- Lancez la commande

```
npm run dev
```

## Consigne

Réalisez chaque exercice en suivant les instructions dans la colonne de droite
